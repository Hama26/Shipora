The repository contains an ArchiMate Archi project (`Shipora.xml`) with all the relevant diagrams for Shipora.

#### How to add your diagrams:
1. Open `Shipora.xml` in ArchiMate Archi
2. Select the model
3. Go to `File -> Import -> Another Model Into Selected Model...`
5. Choose the path of the Archimate project containing your diagram
6. Now, export `Shipora.xml` with the new changes
7. `git add Shipora.xml` and `git commit`
